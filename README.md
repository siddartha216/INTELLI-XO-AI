# 🤖 Intellixo – AI Tic Tac Toe

Intellixo is a smart Tic Tac Toe game powered by Artificial Intelligence.  
It allows users to play against a computer with different difficulty levels using the Minimax algorithm.

---

## 🚀 Features

- 🎮 Interactive Tic Tac Toe game
- 🤖 AI opponent using Minimax algorithm
- 🧠 Three difficulty levels:
  - Easy (random moves)
  - Medium (mix of random + AI)
  - Hard (perfect AI)
- 👤 Player name input
- 🏆 Score tracking system
- ✨ Smooth animations and modern UI
- 🔁 Restart and navigation options
